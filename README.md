# eDashLight Project

- [DEMO](https://taniaomelko.github.io/edashlight/)

## Description

A dashboard that summarizes financial data.

## Local deployment

### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
